# Аудио файлы для пасхалок

## Rammstein Intro для пасхалки Пастернак -> Линдеманн

**Требуется файл:** `rammstein-intro.mp3`

### Где взять:
1. Найти короткое интро из песни Rammstein (5-10 секунд)
2. Конвертировать в MP3 формат
3. Сохранить как `rammstein-intro.mp3` в эту папку

### Рекомендации:
- Длительность: 5-10 секунд
- Формат: MP3
- Битрейт: 128-192 kbps
- Размер: ~100-200 KB

### Альтернативы:
- Можно использовать начало песни "Sonne", "Ich Will" или "Du Hast"
- Или просто гитарное/клавишное интро без вокала

### Как заменить:
Просто положите ваш MP3 файл в эту папку с именем `rammstein-intro.mp3`
