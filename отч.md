Объектом исследования является разработка адаптивного веб-сайта, предоставляющего пользователям доступ к коллекции русской поэзии, биографиям поэтов и интересным фактам о литературе.

Цель работы — создание многофункционального веб-ресурса с удобной навигацией, системой избранного и адаптивным дизайном для различных устройств.

Методы выполнения работы включают использование современных веб-технологий: HTML5, CSS3, JavaScript, Bootstrap 5, jQuery для реализации адаптивных и функциональных интерфейсов с применением компонентного подхода к архитектуре приложения.

В результате работы был создан веб-сайт с адаптивным дизайном, включающий следующие элементы:
- Каталог из 73 стихотворений 16 русских поэтов
- Система поиска и фильтрации стихотворений по различным критериям
- Детальные страницы поэтов с биографиями и статистикой
- Система избранного с локальным хранением данных
- Переключение между светлой и темной темами
- Адаптивный дизайн для мобильных устройств и десктопов
- Функция "Стих дня" с детерминированным выбором
- Компонентная архитектура с динамической загрузкой элементов
- Современный многоколоночный footer с навигацией

Область применения: разработка образовательных и культурных веб-ресурсов с акцентом на сохранение литературного наследия и популяризацию русской поэзии.

В рамках учебного процесса дальнейшее развитие проекта не прогнозируется. Основной целью работы было освоение технологий и методов разработки современных клиентских веб-приложений. Проект завершен в установленном формате.

## 1. ВВЕДЕНИЕ

Современные интернет-ресурсы играют важнейшую роль в сохранении и популяризации культурного наследия. Особое место среди них занимают платформы, посвященные литературе и поэзии. Клиентская часть таких ресурсов должна не только предоставлять доступ к контенту, но и создавать комфортную среду для изучения материала, обеспечивать удобную навигацию и поддерживать интерактивное взаимодействие с пользователем.

Качественная реализация клиентской части напрямую влияет на пользовательский опыт: скорость работы, интуитивность интерфейса, визуальная привлекательность и функциональность определяют успешность достижения целей проекта. Без должного внимания к этим аспектам невозможно обеспечить положительное восприятие ресурса целевой аудиторией.

**Актуальность темы** обусловлена растущей потребностью в цифровизации культурного контента и необходимостью создания современных образовательных платформ, доступных широкой аудитории. Русская поэзия является неотъемлемой частью мирового культурного наследия, и её популяризация через современные цифровые технологии способствует сохранению литературных традиций для будущих поколений.

**Целью данной курсовой работы** является разработка и реализация клиентской части интернет-ресурса «Сборник Стихов» — современной веб-платформы для изучения русской поэзии. В процессе работы использовались актуальные веб-технологии: HTML5 для структурирования контента, CSS3 для стилизации и создания адаптивного дизайна, JavaScript для реализации интерактивной функциональности, а также фреймворки Bootstrap 5 и jQuery для ускорения разработки.

**В соответствии с поставленной целью решаются следующие задачи:**

1. Провести анализ предметной области и изучить существующие аналоги литературных веб-платформ
2. Изучить и выбрать оптимальные технологии для разработки клиентской части веб-приложения
3. Спроектировать архитектуру приложения с применением компонентного подхода
4. Создать адаптивный пользовательский интерфейс с учетом принципов UX/UI дизайна
5. Реализовать систему управления данными с использованием JSON и LocalStorage
6. Разработать функциональные модули: поиск, фильтрация, избранное, темная тема
7. Создать интерактивные элементы для улучшения пользовательского опыта
8. Обеспечить кроссбраузерную совместимость и адаптивность для мобильных устройств
9. Провести тестирование разработанного веб-приложения

**Методы исследования:**
- Анализ существующих литературных веб-платформ и определение требований к функционалу
- Изучение технической документации по веб-технологиям (MDN, W3C)
- Применение методологии компонентно-ориентированной разработки
- Использование методов адаптивной верстки и отзывчивого дизайна
- Тестирование в различных браузерах и на различных устройствах

**Информационная база исследования:**
При разработке курсовой работы использовались знания, полученные в ходе самостоятельного изучения материала, лекционных и практических занятий по курсу «Проектирование и разработка клиентских частей интернет-ресурсов», а также актуальная техническая документация и руководства по современным веб-технологиям.

## 2. ОБЩИЕ СВЕДЕНИЯ

### 2.1 Наименование клиентской части интернет-ресурса

Темой разработанной клиентской части интернет-ресурса является «Сборник Стихов» — образовательная платформа для изучения русской поэзии.

В качестве наименования продукта курсовой работы было выбрано название, соответствующее выбранной теме и отражающее основное назначение ресурса. Разработанный продукт интернет-ресурса имеет официальное название — **«Клиентская часть интернет-ресурса "Сборник Стихов"»**.

Данное название было выбрано исходя из следующих соображений:
- Краткость и запоминаемость
- Четкое отражение тематики ресурса (поэзия, стихотворения)
- Универсальность и понятность для широкой аудитории
- Отсутствие языковых барьеров при восприятии названия

### 2.2 Функциональное назначение

Данное веб-приложение направлено на предоставление пользователям удобного и информативного интерфейса для изучения русской поэзии, знакомства с биографиями поэтов и расширения литературного кругозора.

**Основная идея** веб-приложения заключается в создании современной цифровой библиотеки поэтических произведений с интуитивно понятным интерфейсом, который позволяет пользователям:

1. **Исследовать коллекцию стихотворений:**
   - Просматривать каталог из 73 стихотворений 16 русских поэтов
   - Использовать систему поиска по названию, автору или содержанию текста
   - Фильтровать стихи по настроению, тегам и периодам
   - Сортировать результаты по различным критериям

2. **Изучать биографии поэтов:**
   - Просматривать детальные страницы с информацией о 16 поэтах
   - Знакомиться с биографией, периодом творчества и стилистикой
   - Изучать статистику произведений каждого автора
   - Просматривать список известных произведений

3. **Управлять избранным контентом:**
   - Добавлять понравившиеся стихотворения в избранное
   - Отмечать любимых поэтов
   - Просматривать коллекцию избранного на отдельной странице
   - Данные сохраняются локально в браузере пользователя

4. **Настраивать интерфейс:**
   - Переключаться между светлой и темной темами оформления
   - Пользоваться адаптивным дизайном на любых устройствах
   - Получать уведомления о действиях в системе

**Целевая аудитория:**
- Учащиеся и студенты, изучающие русскую литературу
- Преподаватели литературы и русского языка
- Любители поэзии и литературные энтузиасты
- Исследователи и культурологи
- Широкая аудитория, интересующаяся русской культурой

**Функциональные возможности:**

| Модуль | Описание функционала |
|--------|---------------------|
| Главная страница | Героическая секция с приветствием, кнопка случайного стихотворения, быстрый доступ к основным разделам |
| Каталог стихов | Полный список стихотворений с карточками, содержащими превью текста, информацию об авторе и теги |
| Страница поэтов | Галерея карточек поэтов с фотографиями, годами жизни и кратким описанием |

| Избранное | Персональная коллекция отмеченных стихов и поэтов с возможностью управления |

**Технические особенности:**

Разработанная клиентская часть реализует следующие технические решения:

1. **Компонентная архитектура:**
   - Динамическая загрузка header и footer компонентов
   - Переиспользуемые JavaScript модули (PoemsManager, FavoritesManager)
   - Модульная структура стилей CSS

2. **Управление данными:**
   - Централизованное хранение данных в JSON файлах (poems.json, poets.json, facts.json)
   - Класс PoemsManager для работы с данными
   - LocalStorage для хранения избранного и настроек темы

3. **Производительность:**
   - Ленивая загрузка изображений
   - Минимизация перерисовок DOM
   - Оптимизация запросов к данным

4. **Доступность:**
   - Семантическая разметка HTML5
   - ARIA-атрибуты для скрин-ридеров
   - Поддержка навигации с клавиатуры

**Перспективы развития:**

В будущем возможно расширение функциональности:
- Добавление серверной части для регистрации пользователей
- Интеграция аудио-чтения стихотворений
- Система комментариев и обсуждений

- Экспорт избранного в различные форматы
- Интеграция с социальными сетями
- Мультиязычность интерфейса

Разработанная клиентская часть может быть использована в качестве основы для создания полноценного образовательного портала по русской литературе. Система поддерживает масштабирование и легко адаптируется под различные требования.

```css
  }
```

```javascript
    try {
      const [poemsData, poetsData] = await Promise.all([
        fetch('/data/poems.json').then(r => r.json()),
        fetch('/data/poets.json').then(r => r.json())
      ]);
      
      this.poems = poemsData;
      this.poets = poetsData;

    } catch (error) {

  }

    return this.poems.filter(poem => 
      poem.title.toLowerCase().includes(query.toLowerCase()) ||
      poem.text.toLowerCase().includes(query.toLowerCase())
    );
  }
}
```

```javascript
function pluralize(count, word) {
    const cases = [2, 0, 1, 1, 1, 2];
    const titles = {
        'стихотворение': ['стихотворение', 'стихотворения', 'стихотворений']
    };
    
    return titles[word][
        count % 100 > 4 && count % 100 < 20 
            ? 2 
            : cases[Math.min(count % 10, 5)]
    ];
}
```

```javascript
function setupEventListeners() {
    // Поиск с debounce

// Применение фильтров и сортировки
function applyFilters() {
    const searchQuery = document.getElementById('search-input').value.toLowerCase();
    const moodFilter = document.getElementById('mood-filter').value;
    const sortBy = document.getElementById('sort-select').value;
    
    // Фильтрация
    filteredPoems = allPoems.filter(poem => {
        const matchesSearch = !searchQuery || 
            poem.title.toLowerCase().includes(searchQuery) ||
            poem.author.toLowerCase().includes(searchQuery) ||
            poem.text.toLowerCase().includes(searchQuery);
            
        const matchesMood = !moodFilter || poem.mood === moodFilter;
        
        return matchesSearch && matchesMood;
    });
    
    // Сортировка
    filteredPoems.sort((a, b) => {
        switch(sortBy) {
            case 'title':
                return a.title.localeCompare(b.title, 'ru');
            case 'author':
                return a.author.localeCompare(b.author, 'ru');
            case 'year':
                return a.year - b.year;
            default:
                return 0;
        }
    });
    
    // Отображение активных фильтров
    showActiveFilters();
    
    // Рендер
    renderPoems(filteredPoems);
}
```

### 3.5 Реализация слоя клиентской логики с применением JavaScript

Клиентская логика приложения реализована с использованием объектно-ориентированного подхода и модульной архитектуры.

**3.5.1 Менеджер данных (poems-manager.js)**

Центральный класс для управления данными о стихах и поэтах:

```javascript
class PoemsManager {
    constructor() {
        this.poems = [];
        this.poets = [];
        this.facts = [];
        this.dataLoaded = false;
    }

    // Загрузка всех данных
    async loadData() {
        if (this.dataLoaded) return;
        
        try {
            const [poemsResponse, poetsResponse, factsResponse] = await Promise.all([
                fetch('/data/poems.json'),
                fetch('/data/poets.json'),
                fetch('/data/facts.json')
            ]);

            this.poems = await poemsResponse.json();
            this.poets = await poetsResponse.json();
            this.facts = await factsResponse.json();
            
            this.dataLoaded = true;
            
            // Событие о загрузке данных
            document.dispatchEvent(new CustomEvent('poemsDataLoaded', {
                detail: { poems: this.poems, poets: this.poets }
            }));
            
        } catch (error) {
            console.error('Ошибка загрузки данных:', error);
            throw error;
        }
    }

    // Получение всех стихов
    getAllPoems() {
        return [...this.poems];
    }

    // Получение всех поэтов
    getAllPoets() {
        return [...this.poets];
    }

    // Поиск стихотворения по ID
    getPoemById(id) {
        return this.poems.find(poem => poem.id === parseInt(id));
    }

    // Поиск поэта по ID
    getPoetById(id) {
        return this.poets.find(poet => poet.id === id);
    }

    // Получение стихов определенного автора
    getPoemsByAuthor(authorId) {
        return this.poems.filter(poem => poem.authorId === authorId);
    }

    // Поиск стихов
    searchPoems(query) {
        const lowerQuery = query.toLowerCase();
        
        return this.poems.filter(poem =>
            poem.title.toLowerCase().includes(lowerQuery) ||
            poem.author.toLowerCase().includes(lowerQuery) ||
            poem.text.toLowerCase().includes(lowerQuery) ||
            poem.tags.some(tag => tag.toLowerCase().includes(lowerQuery))
        );
    }

    // Фильтрация стихов
    filterPoems(filters) {
        return this.poems.filter(poem => {
            // Фильтр по автору
            if (filters.author && poem.authorId !== filters.author) {
                return false;
            }
            
            // Фильтр по настроению
            if (filters.mood && poem.mood !== filters.mood) {
                return false;
            }
            
            // Фильтр по тегам
            if (filters.tags && filters.tags.length > 0) {
                const hasTag = filters.tags.some(tag => poem.tags.includes(tag));
                if (!hasTag) return false;
            }
            
            // Фильтр по периоду
            if (filters.yearFrom && poem.year < filters.yearFrom) {
                return false;
            }
            if (filters.yearTo && poem.year > filters.yearTo) {
                return false;
            }
            
            return true;
        });
    }

    // Получение случайного стихотворения
    getRandomPoem() {
        const randomIndex = Math.floor(Math.random() * this.poems.length);
        return this.poems[randomIndex];
    }

    // Статистика
    getStatistics() {
        const poetStats = {};
        
        this.poems.forEach(poem => {
            if (!poetStats[poem.authorId]) {
                poetStats[poem.authorId] = {
                    count: 0,
                    moods: {},
                    years: []
                };
            }
            
            poetStats[poem.authorId].count++;
            poetStats[poem.authorId].moods[poem.mood] = 
                (poetStats[poem.authorId].moods[poem.mood] || 0) + 1;
            poetStats[poem.authorId].years.push(poem.year);
        });
        
        return {
            totalPoems: this.poems.length,
            totalPoets: this.poets.length,
            poetStats: poetStats
        };
    }
}
```

```javascript
class FavoritesManager {
    constructor() {
        this.storageKey = 'poetry-favorites';
        this.favorites = this.loadFromStorage();
    }

    // Загрузка из LocalStorage
    loadFromStorage() {
        try {
            const data = localStorage.getItem(this.storageKey);
            if (data) {
                return JSON.parse(data);
            }
        } catch (error) {
            console.error('Ошибка загрузки избранного:', error);
        }
        
        return {
            poems: [],
            poets: []
        };
    }

    // Сохранение в LocalStorage
    saveToStorage() {
        try {
            localStorage.setItem(this.storageKey, JSON.stringify(this.favorites));
            
            // Событие об изменении избранного
            document.dispatchEvent(new CustomEvent('favoritesChanged', {
                detail: { favorites: this.favorites }
            }));
        } catch (error) {
            console.error('Ошибка сохранения избранного:', error);
        }
    }

    // Переключение избранного
    toggleFavorite(type, id) {
        const array = this.favorites[type];
        const index = array.indexOf(id);
        
        if (index > -1) {
            array.splice(index, 1);
            this.showNotification(`Удалено из избранного`, 'info');
        } else {
            array.push(id);
            this.showNotification(`Добавлено в избранное`, 'success');
        }
        
        this.saveToStorage();
    }

    // Проверка наличия в избранном
    isFavorite(type, id) {
        return this.favorites[type].includes(id);
    }

    // Получение всех избранных
    getAllFavorites() {
        return {
            poems: [...this.favorites.poems],
            poets: [...this.favorites.poets]
        };
    }

    // Очистка избранного
    clearFavorites(type) {
        if (type) {
```

```javascript
class PoemOfTheDay {
    constructor(poemsManager, favoritesManager) {
        this.poemsManager = poemsManager;
        this.favoritesManager = favoritesManager;
    }

    // Генерация хеша на основе даты
    getDateHash(dateString) {
        let hash = 0;
        for (let i = 0; i < dateString.length; i++) {
            const char = dateString.charCodeAt(i);
            hash = ((hash << 5) - hash) + char;
            hash = hash & hash; // Convert to 32bit integer
        }
        return Math.abs(hash);
    }

    // Получение стиха дня
    getPoemOfTheDay() {
        const today = new Date();
        const dateString = `${today.getFullYear()}-${today.getMonth()}-${today.getDate()}`;
        const poems = this.poemsManager.getAllPoems();
        
        if (poems.length === 0) return null;
        
        const hash = this.getDateHash(dateString);
        const index = hash % poems.length;
        
        return poems[index];
    }

    // Отображение стиха дня
    displayPoemOfTheDay(containerId) {
        const poem = this.getPoemOfTheDay();
        if (!poem) return;

        const container = document.getElementById(containerId);
        const isFavorite = this.favoritesManager.isPoemFavorite(poem.id);
        const moodEmoji = this.getMoodEmoji(poem.mood);

        container.innerHTML = `

                <div class="poem-of-day-header">
                    <h3>${poem.title}</h3>
                    <p class="poem-author">— ${poem.author}</p>
                </div>
                <div class="poem-of-day-body">
                    <p class="poem-preview">${poem.text.split('\n').slice(0, 4).join('\n')}...</p>
                </div>
                <div class="poem-of-day-footer">
                    <div class="poem-badges">
                        <span class="mood-badge">${moodEmoji} ${poem.mood}</span>
                        ${poem.tags.map(tag => 
                            `<span class="tag-badge">#${tag}</span>`
                        ).join('')}
                    </div>
                    <div class="poem-actions">
                        <button class="btn btn-sm btn-outline-primary" 
                                onclick="showPoemModal(${poem.id})">
                            Читать полностью
                        </button>

                        </button>
                    </div>
                </div>
            </div>
        `;
    }

    // Эмодзи для настроений

            'меланхоличное': '🌙',
            'радостное': '☀️',
            'философское': '🤔',
            'романтичное': '💕',
            'патриотическое': '🇷🇺',
            'созерцательное': '🧘',
            'трагичное': '😢',
            'героическое': '⚔️'
        };
        return moodEmojis[mood] || '📖';
    }
}
```
